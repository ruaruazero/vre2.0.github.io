Instructions for use

        We have eight search mode to do

        1·Find basic protein structures for each influenza A protein.

        2·Find 3D protein structures by protein name, subtype, strain name or advanced search criteria including public identifiers.

        3·Find Proteins with structures by protein name, subtype, strain name or advanced search criteria including public identifiers.

        4·Find 3D Protein Structures - by PDB ID
        





        
Example

        1. Click Search for Protein with structures files.

        2. Check A in VIRUS TYPE.

        3. Select 4 HA from Proteins to Search.

        4. Click Search
